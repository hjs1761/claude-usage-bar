# Claude Usage Bar — 설치 & 배포 안내

macOS 메뉴바에 Claude **세션(5h)/주간** 사용량을 띄우는 앱입니다.
Claude Code 로그인 토큰을 **읽기 전용**으로 사용해 공식 사용량 API를 조회합니다(제3자 서버 없음).

## 처음 설치 (동료용)

1. **최신 릴리즈 zip 다운로드**
   → https://github.com/hjs1761/claude-usage-bar/releases/latest
   의 `claude-usage-mac-vX.Y.zip`
2. 압축 해제 → **`Claude Usage Bar.app`을 `/응용 프로그램`(`/Applications`)으로** 이동
3. **최초 1회만** 터미널에서 Gatekeeper 격리 해제(개인 서명 앱이라 필요):
   ```bash
   xattr -cr "/Applications/Claude Usage Bar.app"
   ```
4. 실행 → 메뉴바에 사용량이 뜹니다.
   - **Claude Code 로그인 필요**: 터미널에서 `claude` 실행 후 `/login` 되어 있어야 토큰을 읽습니다.

## 업데이트 (설치 이후엔 자동)

- 앱 안 **설정 → [업데이트 확인]** → 새 버전 있으면 **[vX.Y 설치]** 클릭 → 자동 다운로드·교체·재실행.
- 앱이 하루 1회 + 시작 시 자동으로 확인해, 새 버전이 나오면 설정에 설치 버튼이 활성화됩니다.
- ⚠ **최초 1회는 위 "처음 설치"로 수동 설치**해야 합니다(업데이트 기능은 v1.4부터 들어감 —
  그 이전 버전/미설치 상태에선 앱 내 업데이트가 안 뜸). 한 번 최신으로 깔면 이후는 전부 자동.

## 기능

- **세션(5h)/주간 사용량** + 진행바(정상 파랑 / 70%↑ 주황 / 90%↑ 빨강)
- **소진 예측**: `🔥 이 속도면 ~1h20m 후 한도` (계정 스냅샷 기반 → 기기 간 동일)
- **상세 대시보드**: 로컬 로그 기반 토큰/비용 (설정 옆 버튼)
- **문의하기**: 설정 → 문의하기 → 내용 전송 시 담당 Dooray 방으로 자동 게시(앱 버전·OS 자동 첨부)

## 참고 / 문제해결

- **"확인되지 않은 개발자" 경고**: 위 3번 `xattr -cr` 하면 해제. (개인용 ad-hoc 서명)
- **"요청 제한(429)"**: 사용량 API 계정 한도. 잠시 후 자동 복구(여러 사용량 앱을 동시에 많이 켜면 발생).
- **여러 대(맥북·미니 등)에 쓸 때**: 기기마다 위 "처음 설치"를 **한 번씩** 해주세요.
  (빌드된 앱은 기기 간 자동 동기화가 아니라, 각자 릴리즈 zip으로 최초 설치 → 이후 앱 내 업데이트)
- 소스: 이 저장소 · 릴리즈는 태그(`vX.Y`) push 시 GitHub Actions가 자동 빌드/게시.
