# Agent Usage Monitor (macOS)

**Claude Code 사용량을 macOS 메뉴바에서 한눈에** 보여주는 앱입니다.
5시간·주간 한도 사용률과 소진 예측을 메뉴바에서, 일·주·월 토큰과 비용(USD)을 대시보드로 실시간 확인할 수 있습니다.

> 이 앱은 **이 Mac에 설치·로그인된 Claude Code**의 사용량을 읽어 표시합니다.
> 사용률은 Claude 공식 사용량 API에서, 토큰·비용 통계는 이 Mac에 저장된 Claude Code 기록에서 가져옵니다.
> **서버 전송 없음 · 읽기 전용 · 개인정보 수집 없음.**

---

## 주요 기능

- **메뉴바 표시** — 세션(5h)/주간(1W) 사용률 % + 리셋까지 남은 시간 (예: `5h 37% · 3h35m`)
  - 표시 모드 4가지: 순환(5h ⇄ 1W, 기본) / 둘 다 / 5h만 / 1W만
  - 사용률에 따라 색 변화: 정상 / 70%↑ 주황 / 90%↑ 빨강 (라이트·다크 자동 대응)
- **한도 소진 예측** — 최근 사용 속도로 세션 한도 도달 시각 추정 (`🔥 이 속도면 ~1h20m 후 한도 도달`)
- **팝오버 요약** (메뉴바 클릭) — 세션·주간·모델별 한도 진행바, 추가 사용량(extra), 오늘/이번 주/이번 달 토큰·비용
- **상세 대시보드**
  - 이번 달/주/오늘 비용·토큰 요약 카드
  - 비용 추이 차트 (막대 클릭으로 드릴다운, 기간 이동)
  - 모델별 비용 도넛 차트 (Opus / Sonnet / Haiku)
  - 프로젝트별 사용량 랭킹 (**cwd 기준 / 파일 기준** 전환, 클릭 시 프로젝트별 상세)
- **원클릭 자동 업데이트** — 새 버전이 나오면 메뉴바에 🆙 표시, 클릭 한 번으로 설치·재실행
- **다크/라이트 테마 자동 대응** · **로그인 시 자동 실행** · **배터리 절전 모드**
- **문의하기** — 앱에서 바로 담당 채널로 의견 전송 (앱 버전·OS 자동 첨부)

## 설치 전 준비

| 준비물 | 설명 |
|---|---|
| **macOS 14 (Sonoma) 이상** | 별도 런타임 설치가 필요 없습니다. |
| **Claude Code (설치·로그인)** | 이 앱은 Claude Code의 사용량을 보여줍니다. 터미널에서 `claude` 를 실행해 **로그인된 상태**여야 사용률이 표시됩니다. |

> `claude` 로그인을 하면 자동으로 복구됩니다.

## 설치 방법 — GitHub 릴리스에서 내려받기

### 1. 릴리스 페이지 열기

아래 **최신 릴리스** 페이지로 이동합니다:

**👉 https://github.com/Weaverloft-AILabs/Agent-Usage-Monitor-Mac/releases/latest**

페이지 아래 **`Assets`**(자산) 목록을 펼칩니다.

### 2. zip 내려받기

**`agent-usage-monitor-mac-vX.Y.zip`** 을 클릭해 내려받습니다 (`X.Y` = 최신 버전, 약 1.3MB).

### 3. 응용 프로그램으로 이동

내려받은 zip의 압축을 풀면 **`Agent Usage Monitor.app`** 이 나옵니다.
이 앱을 **응용 프로그램(`/Applications`) 폴더**로 드래그해 옮깁니다.

### 4. 최초 1회 — Gatekeeper 격리 해제

이 앱은 아직 Apple 공증(notarize) 서명이 없어, 처음 실행할 때 macOS가 **"확인되지 않은 개발자 / 손상된 파일"** 로 막을 수 있습니다. 아래 중 하나로 해제하세요.

**방법 A (터미널 — 가장 확실):**

```bash
xattr -cr "/Applications/Agent Usage Monitor.app"
```

실행 후 앱을 더블클릭하면 열립니다.

**방법 B (우클릭):**

Finder에서 앱을 **우클릭 → 열기 → (경고창에서) 열기**.
(막히면 방법 A를 사용하세요.)

### 5. 첫 실행

앱을 실행하면 **메뉴바 오른쪽**에 사용량이 나타납니다 (예: `5h 37% · 3h35m`).
- 처음엔 `◵` 로 표시되다가, 잠시 후 실제 사용률로 바뀝니다.
- **Ice·Bartender** 같은 메뉴바 정리 앱을 쓰면 숨김 영역에 들어가 있을 수 있습니다.

---

## 업데이트 (설치 이후엔 자동)

- 앱이 **하루 1회 + 실행 시** 자동으로 새 버전을 확인합니다.
- 새 버전이 있으면 **메뉴바에 🆙**, 팝오버·설정에 **설치 버튼**이 나타납니다.
  버튼을 누르면 **다운로드 → 교체 → 재실행**이 자동으로 진행됩니다.
- ⚠ **최초 1회는 위 "설치 방법"으로 수동 설치**해야 합니다. 한 번 설치하면 이후는 전부 자동입니다.

## 사용법

### 메뉴바

- 사용률 텍스트를 **클릭**하면 요약 팝오버가 열립니다 (세션·주간 한도 진행바, 소진 예측, 오늘/주/월 비용).
- 색: 정상 · 70% 이상 주황 · 90% 이상 빨강. 세션 소진 임박 시 앞에 🔥 가 붙습니다.

### 상세 대시보드

팝오버의 **[상세 대시보드 열기]** 버튼 → 비용 추이·모델별·프로젝트별 통계를 큰 창으로 봅니다.
- **비용 추이** 막대를 클릭하면 하위 기간으로 드릴다운됩니다.
- **프로젝트별**은 `cwd 기준`(작업 디렉터리)과 `파일 기준`(실제 편집한 파일 위치)으로 전환할 수 있습니다.

### 설정

팝오버의 **[설정]** 버튼에서:
- **메뉴바 표시** 모드 (순환 / 둘 다 / 5h만 / 1W만)
- **새로고침 주기** (30초 ~ 10분)
- **충전 연동 절전** (배터리일 때 갱신 간격 완화)
- **로그인 시 자동 실행**
- **업데이트 확인** · **문의하기**

## 문제 해결

| 증상 | 해결 |
|---|---|
| **"확인되지 않은 개발자"로 실행이 막힘** | 설치 4번의 `xattr -cr ...` 실행 (또는 우클릭 → 열기). |
| **"로그인 필요"만 표시됨** | 터미널에서 `claude` 실행 후 로그인 상태 확인. |
| **숫자 대신 `◵` 가 계속 뜸** | 잠시 후 자동 재시도. 여러 사용량 앱을 동시에 많이 켜면 API 요청 제한(429)에 걸릴 수 있으니 잠깐 기다리세요. |
| **메뉴바에서 안 보임** | Ice·Bartender 등 메뉴바 매니저의 숨김 영역 확인. |
| **비용 통계가 비어 있음** | `~/.claude/projects` 에 Claude Code 사용 기록(`*.jsonl`)이 있어야 합니다. Claude Code를 실제로 사용한 뒤 표시됩니다. |

## 개인정보 · 보안

- 사용률은 이 Mac에 **이미 로그인된** Claude Code 토큰을 **읽기 전용**으로만 사용합니다 (별도 로그인 UI 없음).
- 토큰·사용 기록은 **이 Mac 밖으로 전송되지 않습니다.** (제3자 서버 없음)
- 사내 배포용 도구이며, 각 사용자가 **본인 계정**으로만 사용합니다.
